# Sistema de Gestión de Garaje

Aplicación sencilla para registrar el ingreso de vehículos. Proyecto desarrollado en **Python** con **Tkinter** aplicando una estructura modular.

## Contenido del proyecto
* **main.py**: Inicia la aplicación creada.
* **modelos**: Define los datos del vehículo (Placa, Marca, Propietario).
* **servicios**: Sirve para guardar y listar los vehículos.
* **ui**: Contiene la interfaz gráfica (ventana, botones y tabla).

## Cómo usar
Se debe ejecutar para completar los campos del formulario y presionar "Agregar Vehículo".
